# Latin Inferno

Ett interaktivt latinäventyr för gymnasiekursen **Latin – språk och kultur 1** (Skolverket, kurskod **LATLAT01**).

Du spelar Dante, vilse i den mörka skogen, och leds av skalden **Vergilius** — som skrev *Aeneiden* på latin — ned genom helvetets nio kretsar och upp igen till stjärnorna. Vergilius lär dig latin medan ni stiger ned: latinet är bokstavligen guidens eget språk.

## Vad det tränar
Varje krets är en nivå kopplad till kursens centrala innehåll och betygskriterier (E/C/A):
uttal, formlära (deklinationer, konjugationer, kasus), enkel meningsbyggnad, basordförråd
som lever vidare i svenska och engelska, latinets historia samt romersk historia och kultur.
En **betygskompass** visar hur svaren speglar Skolverkets kunskapskrav.

## Äkthet
Dantes *Inferno* (italienska) är berättelsens ram. Allt **latin** är äkta kurslatin, riktiga
rader ur Vergilius *Aeneiden* (med bok- och radhänvisning) och autentiska sentenser — inget påhittat.

## Teknik
Statisk webbapp (HTML/CSS/vanilla JS), ingen inloggning, sparar framsteg lokalt i webbläsaren.
Ligger som GitHub Page. Källinnehåll: `js/data.js` · motor: `js/engine.js` · kursunderlag: `CURRICULUM.md`.

*Svensktledd, tvåspråkig ram (svenska som instruktionsspråk, latin som mål, svensk/engelsk härledning).*
