# Latin – språk och kultur 1 (LATLAT01) — Curriculum reference

Source: Skolverket ämnesplan, ämne **Latin – språk och kultur** (ämneskod **LAT**).
Course: **Latin – språk och kultur 1**, kurskod **LATLAT01**, 100 poäng.
Grade criteria text verified verbatim via betygskriterier.se (Skolverket matris, retrieved 2023-09-07).

> System note (Gy11 vs Gy25): LATLAT01 is the Gy11 course code. From 2025-07-01 Skolverket
> migrated gymnasiet to Gy25 (ämnesbetyg / nivåer). Core Latin-1 content is essentially
> stable across systems. CONFIRM with johan which system the student is in (depends on
> whether they started gymnasium Aug 2024 = Gy11, or Aug 2025 = Gy25).

## Centralt innehåll (core content the course must cover)

1. Läsning och tolkning av mycket enkla latinska texter — texter från läroböcker, mycket
   enkla autentiska texter, citat och sentenser, samt bearbetade autentiska texter (t.ex. fabler).
2. Grunderna i latinets uttal.
3. Latinsk grammatik med grundläggande formlära och enkel meningsbyggnad.
4. Relevanta begrepp som behövs för att beskriva och analysera språket.
5. Latinskt basordförråd med betoning på ord, orddelar, begrepp och uttryck som lever vidare
   i de moderna språken och i vetenskaplig terminologi.
6. Översikt över det latinska språkets historia.
7. Latinets inflytande på svenska, engelska och andra moderna språk.
8. Latinets roll som kulturbärare i ett historiskt perspektiv.
9. Översikt över det romerska rikets politiska historia (rikets utbredning, mötet med
   Hannibal, några kända politiker och kejsare).
10. Romersk kultur, med exempel från konst och mytologi, och typiska drag i romerskt
    samhällsliv (familj, vardagsliv, synen på slavar).
11. Romersk historia, kultur och samhällsliv i texterna och i senare tiders konst och
    litteratur, med betoning på vår egen tid.

## Progression pattern (design key)
The grade ladder is built almost entirely on qualifying adverbs/adjectives:
- Recall/recognize -> **översiktligt / enkla / mycket enkla / med viss säkerhet (begränsat)**  = **E**
- Explain/apply -> **utförligt / enkla->välgrundade / med viss säkerhet**  = **C**
- Analyze/nuance -> **utförligt och nyanserat / välgrundade och nyanserade / med säkerhet**  = **A**

Design implication: every challenge can carry a tiered question (E recall, C explain, A
nuanced analysis), so the same content path serves all three grade levels.
